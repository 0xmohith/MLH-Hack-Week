# MLH Global Hack Week (GHW) Projects

*Welcome to my repository for MLH Global Hack Week (GHW)!*
This repo contains a collection of projects, experiments, and mini-builds that I worked on during the hack week.

### About

MLH Global Hack Week is a week-long event organized by Major League Hacking where developers from around the world collaborate, learn, and build projects.

This repository showcases my hands-on learning journey through:
* Building small web applications
* Exploring new technologies
* Practicing problem-solving
* Shipping projects quickly

### Tech Stack

Projects in this repo may include:
* HTML, CSS, JavaScript
* Basic backend (Node.js)
* Git & GitHub
* Other tools explored during the week

### AI Assistance

Some parts of these projects were built with the help of AI tools, including:
* ChatGPT
* GitHub Copilot

These tools were used for:
* Code suggestions
* Debugging help
* Learning new concepts
* Improving productivity

All implementations were reviewed, tested, and understood by me as part of my learning process.
