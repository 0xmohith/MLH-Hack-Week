# PASSWORD STRENGTH CHECKER

### OVERVIEW
A simple web application that evaluates the strength of passwords and gives a relative feedback.
The tool analyzes different password characteristics.

### FEATURES
* Checks minimum password length (at least 8 characters).
* Checks for at least one upper case and one lower case character.
* Checks for special characters and numbers.
* Checks for any blank spaces.
* Warns if the password is commonly used.
* Displays password strength feedback.

### TECHNOLOGIES USED
* HTML
* CSS
* JavaScript
