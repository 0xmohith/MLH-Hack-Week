# Web Scraping & Automation (Cloud Week)

As part of **MLH Global Hack Week – Cloud Week** by Major League Hacking, I worked on simple automation challenges using ***GAFFA***.

These tasks focused on understanding how to automate browser actions and extract data efficiently.

## Automation Tasks & Outputs

### 1. HTML Report to PDF Automation
>Image: gaffa1.png

Executed a request to convert a sample web page into a structured PDF report using GAFFA’s “Print HTML to PDF” functionality.
Demonstrates basic data extraction and automated report generation.

### 2. Browser Automation Workflow
>Image: gaffa2.png

Automated a browser session for the challenge page: https://demo.gaffa.dev/mlh/challenge
The workflow includes scrolling through the page, entering an email ID, and submitting the form automatically.

### 3. Successful Submission Output
>Image: gaffa2.1.png

Captured output from the automated process showing a confirmation pop-up indicating successful form submission.
Validates that the automation flow executed correctly end-to-end.

## Key Takeaways
* Gained hands-on exposure to browser automation.
* Learned how to structure simple automation workflows.
* Understood practical use cases of web scraping and task automation.
* Built confidence in using new tools quickly during hackathon challenges.
